Neste site foi possível treinar conhecimentos fundamentais em REACT + NPM/NPX + REACT SASS + REACT JSON SERVER + REACT AXIOS + REACT ROUTER.

## ⚙ Configuração

1- Para instalar as dependências:

> npm i

2- Para iniciar a aplicação:

> npm start
